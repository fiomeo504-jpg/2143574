# Random Quote Generator — Генератор случайных цитат

**Автор:** Кузнецов Даниил Юрьевич

## Описание программы

GUI-приложение для генерации вдохновляющих и мотивирующих цитат. Позволяет добавлять новые цитаты, просматривать историю сгенерированных цитат и фильтровать их по автору и теме.

### Функциональность:
- **Генерация случайных цитат** из библиотеки
- **Добавление новых цитат** (текст, автор, тема)
- **Проверка на пустые строки** при добавлении
- **История сгенерированных цитат** с датой и временем
- **Фильтрация истории** по автору и теме
- **Сохранение и загрузка** истории в JSON-файл
- **Очистка истории**
- **Статистика** (количество цитат в библиотеке и истории)
