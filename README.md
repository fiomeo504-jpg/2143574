# 🎲 Random Task Generator

**Автор:** Кузнецов Даниил Юрьевич

## 📌 О проекте

**Random Task Generator** — это программа, которая помогает генерировать случайные задачи для учёбы, спорта и работы. Можно также добавлять свои задачи и просматривать историю.

### Основные функции:
- 🎲 **Генерация случайной задачи** из готового списка
- ➕ **Добавление своей задачи** с выбором типа
- 🔍 **Фильтрация истории** по типу задачи
- 💾 **Сохранение истории** в файл JSON
- 📂 **Автоматическая загрузка** истории при запуске
- ✅ **Проверка ввода** (нельзя добавить пустую или слишком длинную задачу)
